Hugo

What is Hugo? Hugo is a fast and simple framework written in Go language to build general purpose websites. There are a lot of pre-buit themes you can choose from and tweak as you go on. A general idea of how Hugo works Create a new site hugo new <site_name>. Clone a theme repository/copy theme files into the site_name/themes folder and add the line theme = "<theme_name>" into config.
