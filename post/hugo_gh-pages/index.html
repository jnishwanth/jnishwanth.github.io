Hugo

What is Hugo?

  • Hugo is a fast and simple framework written in Go language to build general purpose websites.

  • There are a lot of pre-buit themes you can choose from and tweak as you go on.

A general idea of how Hugo works

  • Create a new site using#+BEGIN_SRC shell 

    hugo new site <site_name>
    .

  • Clone a theme repository/copy theme files into the 

    site_name/themes} folder and add  src_toml{theme = <theme_name>} into src_shell{site_name/config.toml
    .

  • Adding posts and other content is using 

    hugo new post/<postname.file>
    , the file is usually a markdown file.

  • Build the site using 

    hugo}. The files are then put into src_shell{site_name/public
    .
