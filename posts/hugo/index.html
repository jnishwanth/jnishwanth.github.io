Hugo

Jan. 19, 2022

What is Hugo?

  • Hugo is a fast and simple framework written in Go language to build general purpose websites.
  • There are a lot of pre-buit themes you can choose from and tweak as you go on.

A general idea of how Hugo works

  • Create a new site hugo new <site_name>.
  • Clone a theme repository/copy theme files into the site_name/themes folder and add the line theme = "<theme_name>" into config.toml.
  • Adding posts and other content is using hugo new posts/<post_name.md> the file is usually a markdown file.
  • Build the site using hugo. The files are then put into site_name/public.

Some notable things of interest

  • Forking and tweaking the pre-built theme you used initially
  • Deploying your site, for instance, like this one, to github pages. (refer this)
